This is code for the NASA Floating Dragon project. It is a system of 2 devices. 1 ESP32 acts a ground station which recieves the flight data being transmitted, organizes it, and prints to the Serial Monitor. Another ESP32 interfaces with a Adafruit BNO055, Adafruit BME688, and 2 Adafruit DRV8871 DC Motor Drivers to form the avionics for the Aerial Descent vehicle. The Avionics is powered using a 14.4v battery and a USB Power Bank.
